# Original Stitch x Pokemon Statistics Tool
## Interactive Statistics Analysis Tool for "Original Stitch x Pokemon" Designs
## 

### What is this?
#
A particular professor of mine uses Original Stitch Pokemon Designs as his desktop background, running through them every time the laptop is booted, wakes up, or after a fixed amount of time has passed. My class has developed a game around this where we all try and guess which pokemon will appear at the beginning of each lecture.

This is a tool to calculate the odds of any given pokemon appearing, factoring in pokemon that have multiple prints and prints which feature multiple pokemon. I've included functionality to show the odds for pokemon types and generation as well, all of which is sortable by any or all of these properties. There is also search functionality and a feature that allows users to randomly pull from the dataset, weighted to account for pokemon with multiple appearances, and stylized as a roulette wheel to fit the Gambling in Vegas aesthetic.

### Original Stitch
#

Original Stitch was a Japanese company that produced textile products (clothes, masks, bandanas, drawstring purses, pillow cases, etc.) that used custom design patterns featuring characters from popular Japanese media. Their two largest and most prominent collaborations were with Pokemon and One Piece, but they also produced smaller sets with other IPs.
They were best known for their custom shirts, which could be HIGHLY customized. For example, an order for a dress shirt would specify...
* Fabric
    * Base
    * Outer Coller
    * Inner Collar
    * Outer Cuff
    * Inner Cuff
    * Pocket
    * Outer Placket
* Style
    * Gender Fitting (2 Options)
    * Collar Style (5 Options)
    * Sleeve Style (2 Options)
    * Button Color (9 Options)
        * Can choose the specific color of every individual button
    * Pocket Position (3 Options, including no pocket)
* Sizing
    * Base Size (6 Options, XS-XL)
    * Collar Size (18 Options, 14"-22.5")
    * Sleeve Length (26 Options, 22.5"-39")

This is all to say that this company made outstanding, unique, customizable products of incredibly high quality. It's a shame they're not around anymore, but before they disappeared, they released all of their designs to the public, in full resolution.
